Introdução

Este tutorial irá guiá-lo na criação de uma página web simples usando HTML, CSS e JavaScript.

HTML

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web.

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <title>Minha Página</title>
</head>
<body>
  <h1>Olá, Mundo!</h1>
</body>
</html>

CSS

CSS (Cascading Style Sheets) é utilizado para estilizar a página web.

body {
  background-color: lightblue;
  font-family: Arial, sans-serif;
}
h1 {
  color: navy;
  text-align: center;
}

JavaScript

JavaScript é uma linguagem de programação que permite adicionar interatividade à sua página web.

function saudacao() {
  alert("Olá, Bem-vindo à minha página!");
}
saudacao();
