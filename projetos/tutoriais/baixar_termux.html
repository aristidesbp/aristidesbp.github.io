Passo 1: Baixar o Termux do GitHub

Para baixar a versão mais recente do Termux diretamente do GitHub, siga os passos abaixo:

  • Visite a página oficial do Termux no GitHub: Termux no GitHub
  • Na página de releases, localize a versão mais recente do Termux para o seu dispositivo Android.
  • Baixe o arquivo APK da versão mais recente.
  • Após o download, abra o arquivo APK para iniciar a instalação.

Passo 2: Permitir a Instalação de Aplicativos de Fontes Desconhecidas

Antes de instalar o Termux, você precisará permitir a instalação de aplicativos de fontes desconhecidas. Siga os passos abaixo:

  • Abra as configurações do seu dispositivo Android.
  • Vá para Segurança ou Aplicativos (isso pode variar dependendo do seu dispositivo).
  • Ative a opção Instalar aplicativos de fontes desconhecidas para o navegador ou gerenciador de arquivos que você usou para baixar o APK.

Agora você poderá instalar o Termux sem problemas.

Passo 3: Instalar o Termux

Depois de baixar o APK e permitir a instalação, siga esses passos:

  • Abra o arquivo APK que você baixou.
  • Clique em "Instalar" para iniciar o processo de instalação.
  • Aguarde a conclusão da instalação.

Passo 4: Atualizar o Termux

Após a instalação, é importante atualizar os pacotes do Termux para garantir que você tenha a versão mais recente do sistema e dos aplicativos.
