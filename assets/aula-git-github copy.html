Tutorial de Git e GitHub pelo Terminal

Instalação do Git

Para instalar o Git, siga os passos abaixo de acordo com o seu sistema operacional:

Windows

  1. Baixe o instalador do Git em git-scm.com/download/win.
  2. Execute o instalador e siga as instruções na tela.

macOS

  1. Abra o Terminal.
  2. Digite xcode-select --install e pressione Enter.
  3. Siga as instruções na tela para instalar o Git.

Linux

  1. Abra o Terminal.
  2. Digite sudo apt-get install git (para distribuições baseadas em Debian) ou sudo yum install git (para distribuições baseadas em Red Hat) e pressione Enter.

Configuração Inicial

Após a instalação, configure seu nome de usuário e email:

git config --global user.name "Seu Nome"
    git config --global user.email "seuemail@example.com"

Comandos Básicos

Inicializar um Repositório

git init

Clonar um Repositório

git clone https://github.com/usuario/repo.git

Verificar o Status

git status

Adicionar Arquivos

git add nome_do_arquivo

Fazer Commit

git commit -m "Mensagem do commit"

Enviar Alterações

git push origin branch

Recursos Avançados

Branches

Para criar uma nova branch:

git checkout -b nome_da_branch

Para mudar para uma branch existente:

git checkout nome_da_branch

Mesclagem (Merge)

Para mesclar uma branch na branch atual:

git merge nome_da_branch

Rebase

Para fazer rebase de uma branch:

git rebase nome_da_branch

Stash

Para salvar mudanças temporariamente:

git stash

Para aplicar mudanças salvas:

git stash apply

Conclusão

Este tutorial cobriu os comandos básicos e alguns recursos avançados do Git e GitHub. Para mais informações, consulte a documentação oficial do Git.
