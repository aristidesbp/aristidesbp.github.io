Mini Curso de Termux

O que é o Termux?

Termux é um terminal Linux para Android que permite executar comandos de terminal, instalar pacotes Linux, programar e até configurar servidores diretamente no seu celular.

1. Como Instalar o Termux

Siga os passos abaixo para instalar o Termux em seu dispositivo Android:

  1. Acesse a página do Termux no F-Droid.
  2. Baixe e instale o F-Droid (se ainda não tiver).
  3. Dentro do F-Droid, procure por "Termux" e instale o aplicativo.

2. Configuração Inicial do Termux

Após instalar, abra o Termux e execute os seguintes comandos para atualizar e dar acesso ao armazenamento:

pkg update -y && pkg upgrade -y
termux-setup-storage

Este último comando solicita permissão para acessar o armazenamento do dispositivo. Aceite a solicitação.

3. Comandos Gerais do Termux

Abaixo, alguns comandos básicos para utilizar no Termux:

Atualizar pacotes

pkg update -y && pkg upgrade -y

Criar e navegar por diretórios

mkdir nome_da_pasta   # Criar uma pasta
cd nome_da_pasta       # Entrar na pasta
cd ..                  # Voltar para a pasta anterior

Listar arquivos

ls                    # Lista arquivos e pastas
ls -la                 # Lista com detalhes

Excluir arquivos e pastas

rm nome_do_arquivo    # Deletar um arquivo
rm -r nome_da_pasta    # Deletar uma pasta e seu conteúdo

Editar arquivos

nano nome_do_arquivo  # Abre o editor de texto Nano

Instalar pacotes

pkg install nome_do_pacote  # Ex: pkg install git

4. Exemplos de Utilização

Exemplos de comandos que podem ser úteis para quem está começando:

Instalar Node.js

pkg install nodejs

Criar um arquivo JavaScript e rodá-lo

echo "console.log('Hello, Termux!')" > hello.js
node hello.js

O comando acima cria o arquivo hello.js com uma linha de código que exibe uma mensagem e depois executa o arquivo com o Node.js.

5. Dicas Finais

  • Use Ctrl + C para interromper um processo em execução.
  • Experimente explorar a documentação oficial do Termux para descobrir mais funcionalidades!

Este é um mini curso básico de Termux para iniciantes. Explore e experimente novos comandos para aprender mais!
