Como Instalar Termux e yt-dlp no Android

Passo 1: Instalando o Termux

O Termux é um terminal para Android que permite executar comandos de Linux. Para instalá-lo, siga os passos abaixo:

  1. Acesse a página oficial do Termux no F-Droid.
  2. Baixe e instale o aplicativo em seu dispositivo Android.
  3. Abra o Termux e execute o seguinte comando para atualizar os pacotes: 
    pkg update && pkg upgrade

Passo 2: Instalando o yt-dlp

O yt-dlp é uma ferramenta para baixar vídeos de plataformas como YouTube. Para instalá-lo no Termux, use os comandos abaixo:

  1. Instale o Python e o pip: 
    pkg install python
  2. Instale o yt-dlp usando o pip: 
    pip install -U yt-dlp
  3. Verifique se a instalação foi concluída: 
    yt-dlp --version

Principais Comandos do yt-dlp

Seguem alguns comandos úteis para utilizar o yt-dlp:

  1. Listar arquivos no diretório: 
     ls
  2. Mover arquivos para a pasta de downloads do Android: 
    mv ~/downloads/*.mkv ~/storage/downloads/
  3. Baixar um vídeo ao vivo do YouTube com qualidade máxima de 720p: 
    ~/downloads $ yt-dlp -f 'bestvideo[height<=720]+bestaudio' -o '~/downloads/%(title)s.%(ext)s' [URL do vídeo]

Dicas Finais

Certifique-se de ter espaço suficiente no armazenamento do dispositivo antes de baixar vídeos grandes. Além disso, sempre execute o comando yt-dlp --update para manter a ferramenta atualizada.
