Guia de Instalação e Criação de API com Node.js no Termux

1. Preparação do Ambiente no Termux

Para programar usando Node.js no Termux, siga os passos abaixo:

  1. Abra o Termux e instale o Node.js com o comando: 
    pkg install nodejs
  2. Verifique se o Node.js foi instalado corretamente: 
    node -v
  3. Crie um diretório para o projeto e acesse-o: 
    mkdir meu_api_js && cd meu_api_js
  4. Inicialize o projeto e crie um package.json com o comando: 
    npm init -y
  5. Instale o Express, o framework para APIs: 
    npm install express

2. Criando a API com Express

Com o ambiente configurado, vamos criar uma API simples.

1. No diretório do projeto, crie um arquivo app.js:

nano app.js

2. Adicione o código abaixo para configurar uma API básica:

            
const express = require('express');
const app = express();
const port = 3000;

// Rota principal
app.get('/', (req, res) => {
    res.send('API está funcionando!');
});

// Rota de dados
app.get('/dados', (req, res) => {
    res.json({ nome: 'João', idade: 30 });
});

// Iniciar o servidor
app.listen(port, () => {
    console.log(`API rodando em http://localhost:${port}`);
});

3. Executando a API

Para iniciar o servidor da API, use o seguinte comando:

node app.js

A API estará disponível em http://localhost:3000.

4. Exemplos Práticos de Uso do Node.js

Exemplo: Renomeando Arquivos com Node.js

Este script usa Node.js para renomear arquivos em uma pasta:

            
const fs = require('fs');
const path = require('path');

const pasta = './arquivos';
const novoPrefixo = 'novo_';

fs.readdir(pasta, (err, arquivos) => {
    if (err) return console.error('Erro ao ler a pasta:', err);

    arquivos.forEach(arquivo => {
        const novoNome = novoPrefixo + arquivo;
        const caminhoAntigo = path.join(pasta, arquivo);
        const caminhoNovo = path.join(pasta, novoNome);

        fs.rename(caminhoAntigo, caminhoNovo, err => {
            if (err) console.error('Erro ao renomear o arquivo:', err);
            else console.log(`Renomeado: ${arquivo} -> ${novoNome}`);
        });
    });
});

Exemplo: Fazendo Requisições para APIs Externas

Para trabalhar com APIs externas, instale o pacote axios:

npm install axios

Em seguida, use o código a seguir para buscar dados de uma API:

            
const axios = require('axios');

const url = 'https://jsonplaceholder.typicode.com/users';

axios.get(url)
    .then(response => {
        console.log('Dados recebidos:', response.data);
    })
    .catch(error => {
        console.error('Erro ao buscar dados:', error);
    });
